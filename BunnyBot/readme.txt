Before running, edit the config file to reflect your nabaztag settings:

SerialID	- Your nabaztag serial ID, usually the MAC address
TokenID		- Your nabaztag token ID, obtained from violet - http://my.violet.net
MSNUsername
MSNPassword	- MSN account details to use. To sign up for a new one - https://signup.live.com

To run the console, simply launch the executable.
To install the service, run the install.bat file (as an administrator) and follow the instructions.

Refer to the project wiki for supported commands:
http://code.google.com/p/bunnybot/
